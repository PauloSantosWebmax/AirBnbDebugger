# AirBnbDebugger
AirBnb property debugger for `testing`

Install packages:
```
// install concurrently
npm install

// install express dependencies
npm run install_api_packages

// install vue dependencies
npm run install_vue_packages
```

Run project:
```
// run...
npm run dev
```

Then go to: http://localhost:8080/
